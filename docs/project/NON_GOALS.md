# Non-goals

The v0.1 project does not attempt to:

- replace JSON, JSON-LD, RDF, Croissant, MCP, CloudEvents, PROV-O, OpenAPI, or JSON Schema;
- define a universal ontology of all human knowledge;
- understand opaque data with certainty when no context exists;
- guarantee data truth or quality;
- prove causality;
- resolve every entity automatically;
- enforce legal permissions by itself;
- represent private model chain-of-thought;
- support PDF, image, audio, video, SQL, or live streams in the first MVP;
- build the anticipation engine before Core interoperability is demonstrated;
- claim that ADUC is an industry standard before independent adoption exists.

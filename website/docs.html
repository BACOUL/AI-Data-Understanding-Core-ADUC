Try in 5 minutes

Clone. Validate. Compare.

The current reference tools run locally with Python. Start with the official complete Core contract, then inspect deterministic reports.

01 Clone02 Install03 Validate04 Compare
01

Clone and install

git clone https://github.com/BACOUL/AI-Data-Understanding-Core-ADUC.git
cd AI-Data-Understanding-Core-ADUC
python -m pip install -r requirements-dev.txt
02

Validate the complete Core example

python tools/aduc_core.py validate examples/core/complete-model.example.json
03

Compare two contracts

python tools/aduc_core.py compare examples/core/complete-model.example.json examples/core/complete-model.example.json
04

Run the principal checks

python tools/validate_contracts.py
python -m unittest discover -s tests -p "test_*.py"
python tools/validate_website.py
